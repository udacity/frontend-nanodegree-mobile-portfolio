Добро пожаловать на GitHub Pages. Это автоматический генератор страница самый простой способ создания красивых страниц для всех ваших проектов. Автор свой Страница содержания здесь используя GitHub ароматный Markdown, выберите шаблон созданный дизайнером, и опубликовать. После того, страница генерируется, вы можете проверить новый GH-pages филиал локально. Если вы используете GitHub Desktop, просто синхронизировать ваш репозиторий и вы увидите новую ветвь. ### Шаблоны Дизайнерские Мы созданного некоторые красивые шаблоны для вас использовать. Идите вперед и нажмите "Продолжить в макеты", чтобы просмотреть их. Вы можете легко вернуться к редактировать страницу перед публикацией. После публикации вашей страницы, вы можете вернуться генератор страницы и переключиться на другую тему. Ваше содержание страницы будут сохранены. ### Создание страниц вручную Если вы предпочитаете не использовать автоматический генератор, нажать филиал с именем GH-pages в репозиторий для создания страницы вручную. В дополнение к поддержке регулярных содержание HTML, GitHub Страницы поддержки Джекил, простой, блог известно генератор статического сайта. Джекил позволяет легко создать заголовки веб-узле и нижние колонтитулы без необходимости копировать их по каждой странице. Он также предлагает интеллектуальную поддержку блога и другие расширенные функции шаблонов. ### Авторов и участников Вы можетеmention имя пользователя GitHub, чтобы создать ссылку на свой профиль.
